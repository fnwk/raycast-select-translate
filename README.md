# Select & Translate

Quickly translate selected text from any application using Raycast Translator.

## Features

- Grab selected text from any macOS application
- Automatically open Raycast Translator with the selected text pre-filled
- Works with any text selection system-wide

## Usage

1. Select any text in any application
2. Trigger the "Select & Translate" command (via Raycast or a hotkey)
3. Raycast Translator opens with your selected text ready to translate

## Tips

- Assign a global hotkey to this command for quick access (Raycast Settings > Extensions > Select & Translate > Configure Command)
